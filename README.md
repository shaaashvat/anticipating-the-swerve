> Shifting from reactive collision avoidance to proactive risk mitigation by linking static road infrastructure to dynamic human behavior.

This repository contains the official implementation and research framework for developing a context-aware Advanced Driver-Assistance System (ADAS), bridging the gap between orderly, lane-disciplined traffic and heterogeneous, unstructured environments.

## 🔬 Core Architecture
1. **Perception Module:** Vision-LiDAR fusion for 3D hazard characterization (combining RDD2022 and IDS-JODHA data).
2. **Prediction Module:** A hybrid GCN-Transformer architecture to predict evasive maneuvers.
3. **Validation Framework:** Integrated testing using a custom two-way interactive driving simulator.
